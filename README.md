# NSZ 转 NSP 工具

一个简单的 Python 工具，用于将 Nintendo Switch 游戏的 NSZ 压缩文件批量转换为 NSP 格式。转换完成后会自动删除原始的 NSZ 文件以节省存储空间。

## 功能特点

- 支持批量转换目录下的所有 NSZ 文件
- 图形化界面选择目录
- 自动删除已转换的 NSZ 文件
- 如果发现同名 NSP 文件已存在，则直接删除 NSZ 文件

## 系统要求

- Python 3.6 或更高版本
- NSZ 命令行工具

## 安装步骤

1. 确保已安装 Python 3.6+
2. 安装必要的依赖：
```bash
pip install nsz
```

## 使用方法

1. 运行脚本：
```bash
python nsz2nsp.py
```
2. 在弹出的文件选择对话框中选择包含 NSZ 文件的目录
3. 程序会自动开始转换目录下的所有 NSZ 文件
4. 转换过程中可以在控制台查看进度

## 注意事项

- 转换过程会删除原始的 NSZ 文件，请确保有备份
- 如果目录中已存在同名的 NSP 文件，程序会直接删除对应的 NSZ 文件
- 确保有足够的磁盘空间用于转换操作

## 常见问题

如果遇到 "请先安装 nsz 工具" 的提示，请执行：
```bash
pip install nsz
```
